# Project 50 Tracker

Una aplicación móvil para el "Project 50 Challenge", construida con React Native (Expo).

## Comenzando

1.  **Instalar Dependencias**:
    ```bash
    npm install
    ```

2.  **Correr la App**:
    ```bash
    npx expo start
    ```
    - Escaneá el código QR con tu celular (usando la app Expo Go).
    - O presioná `w` para correrla en el navegador web.

## Estructura del Proyecto

-   `src/app`: Pantallas y Navegación (Expo Router).
-   `src/components`: Componentes UI reutilizables.
-   `src/constants`: Tema, colores y configuración.
-   `src/hooks`: Hooks personalizados de React.
-   `src/store`: Gestión de estado (Zustand).
